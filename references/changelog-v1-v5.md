# 实战经验教训（南京模板版）

旧模板实战经验已完成归档与清理；当前项目以南京城市沙龙模板为唯一有效品牌资产。

仍然保留的通用经验：

1. 背景替换必须使用“添加全屏图片 + z-order 置底”，不要依赖 python-pptx 不存在的背景替换 API。
2. 迁移已有 PPT 时优先使用 `--mode full`，确保背景、字体、配色都被统一处理。
3. 不要用整页不透明大卡遮挡模板背景；内容卡片之间应保留间隙。
4. 运行后必须查看适配报告中的背景替换、颜色替换、对比度修复和安全区警告。
5. HTML/Markdown 生成后必须运行 `scripts/verify_output.py --strict`。
6. 配色必须收敛到南京限定 7 色：`#3272DC`、`#08194B`、`#FFFFFF`、`#00C8D8`、`#01A4FF`、`#44474F`、`#8B8C8C`。
